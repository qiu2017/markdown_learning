具体内容如下
# 一 标题
用#号在前表示标题，1到6个#号分别表示1级到6级标题

另，一级标题可以在标题文本下另起一行用=号表示，二级标题文本下另起一行用-号表示

# 二 段落格式
## 段落换行
方法一 末尾添加两个空格  

方法二 使用空行来换行  

## 字体  
*斜体文本* _斜体文本_

**粗体文本** __粗体文本__

***粗斜体文本*** ___粗斜体文本___

## 分隔线
一行三个以上星号、减号、底线 表示分割线
***
** *
* * *
* * * * *
- - -
--------
______

## 删除线
~~Markdown~~

## 下划线  
<u>带下划线文本</u>

## 注脚  
一行字^[注脚内容]  

## 注释  
*[注释]:注释显示的内容 注意该行引号要小写。

# 三 列表
**无序列表**用星号、加号、减号作为列表标记  
* 第一项
+ 第二项
- 第三项

**有序列表**使用数字并加上.号作为列表标记  
1. 第一项
2. 第二项
3. 第三项

**列表嵌套**只需在子列表的选项前添加四个空格  
1. 人生
    2. 第一件事
        + 吃饭
        + 睡觉

# 四 区块
区块引用是在段落开头使用>符号，可多个嵌套使用

* 开始
    > 最外层
    >> 第一层嵌套
    >> 1. 可以复合使用区块和列表
    >> 2. 要在离别项目内放进区块，要在>符号前添加四个空格
    >>> 第二层嵌套
    >>> + 我有讲明白吗

# 五 代码
1. 一行中用反引号括起来  
`printf()`函数
2. 缩进式插入多行代码  
代码区块使用4个空格或一个制表符（Tab键），代码上方必须有空行


    #include <stdio.h>
    int main(){
        return 0;
    }

3. 用` ``` `（三个重点符）起一行（后面可以说明一种语言）包裹一段代码，并以` ``` `结束

# 六 链接

[链接名称](链接地址)  
或  
<链接地址>  

链接也可以用变量来代替，在文档后面附带变量地址  
这个链接用 1 作为网址变量 [Google][1]

[1]: http://www.google.com/

# 七 图片
![alt 属性文本](图片地址)  
或  
![alt 属性文本](图片地址 "可选标题")  

示例  
![](http://static.runoob.com/images/runoob-logo.png "RUNOOB")

# 八 表格
使用|来分隔不同的单元格，使用-来分隔表头和其他行  
搭配使用:和-设置内容和标题栏对齐方式

|左对齐|右对齐|居中对齐|
|:-|-:|:-:|
|单元格|单元格|单元格|

# 九 高级技巧
不在 Markdown涵盖范围之内的标签，都可以直接在文档里面用 HTML 撰写。

**转义** 使用反斜杠\转义特定字符

\   反斜线
`   反引号
*   星号
_   下划线
{}  花括号
[]  方括号
()  小括号
#   井字号
+   加号
-   减号
.   英文句点
!   感叹号


**数学公式** 略
